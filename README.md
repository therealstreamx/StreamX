# StreamX Media Player for Android

## Overview

**StreamX** is a lightweight media player for Android phones designed for smooth playback of user-provided media sources.

Add your own streams or media sources and StreamX handles the playback experience with a clean, fast interface optimized for mobile.

**Important:** StreamX does **not** host, provide, sell, distribute, or include any content, channels, IPTV services, or playlists. Users are responsible for supplying and managing their own content sources.

---

## Features

* Smooth video playback
* Mobile-friendly interface
* Support for user-added media sources
* Fast loading and responsive playback
* Lightweight install size
* Built for Android phones
* Simple setup and easy navigation

## Installation

### APK Installation

1. Download the StreamX APK
2. Open the APK on your Android phone
3. Allow installation from unknown sources if prompted
4. Complete installation
5. Open StreamX

## Getting Started

1. Launch StreamX
2. Add your own media source or playlist
3. Select your content
4. Start playback

## Requirements

* Android device running Android 8.0+
